SET

-Definition-

A set is a collection of well defined and unique elements. Sets are generally denoted by capital letters and the elements of a set are cased within a pair of curly brackets.

Example: A = {1,2,3} is a set because each of the element is unique and well defined however A = {1, 2, 2, 3} is not a set because the element 2 is repeated, and therefore the elements are no longer unique to each other.

-Important symbols to remember in set-


i. [∈] = Belongs to;

ii. [| or :] = Such that;

iii. [⊆] = Contained in / Subset of;

iv. [,] = And;

v. [∀] = For all / For every;

-Representation of set-

i. Roaster/Tabular form: A = {1,2,3,4,5}

ii. Set builder/Property form: A = {x : 0 < x < 6, x ∈ }

-Types of sets-

i. Finite sets: Sets with defined end point. Example: {x : 0 < x < 5}
